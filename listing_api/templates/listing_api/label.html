Run the eval

After labeling a batch, run the scoring pipeline from the command line: 

python listing_api/eval/run_eval.py --out listing_api/eval/results/run.md

It'll call the live API on every item, score each field against the ground truth you typed here, and write a markdown report with per-field accuracy tables.
